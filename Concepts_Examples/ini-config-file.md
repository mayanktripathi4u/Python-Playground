# INI Config File
An `.ini` file is a simple, text-based configuration file format used to store settings and configuration information. It is commonly used in various applications and programming languages, including Python. The format is straightforward and human-readable, making it an excellent choice for configuration purposes.

# Basic Concepts of `.ini` Files
1. Structure:

* Sections: Denoted by square brackets ([ ]). Each section contains related settings.
Keys and Values: Each setting within a section is represented as a key-value pair, separated by an equal sign (=).
