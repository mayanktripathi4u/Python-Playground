# How to run
Terminal-1: Backend (Run FastAPI)
```bash
# On Terminal # 1
cd 

pip install -r requirements.txt

uvicorn main:app --reload
```

Terminal-2: Frontend (Run Streamlit)
```bash
# On Terminal # 2
cd 

pip install -r requirements.txt

streamlit run app.py 
```
